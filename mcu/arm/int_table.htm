ARM

中斷表

ARM的中斷向量是固定的，因此，只有寫組合語言或反組譯才需要使用，司徒先整理一下列表，以後可以當參考用

ARMv7

位址 名稱
0x00000000 硬體重置
0x00000004 未定義指令
0x00000008 軟體中斷指令
0x0000000C 預先存取中止
0x00000010 資料中止
0x00000014 保留
0x00000018 IRQ
0x0000001C FIQ

Cortex-M3

位址 名稱
0x00000000 Top of Stack
0x00000004 Reset Handler
0x00000008 NMI Handler
0x0000000C Hard Fault Handler
0x00000010 MPU Fault Handler
0x00000014 Usage Fault Handler
