LinkIt Smart MT7688

Build UBoot

UART(57600 bps)
MT7688 P8 => RX
MT7688 P9 => TX 


$ git clone https://github.com/steward-fu/gh_mt7688_toolchain.git
$ sudo cp -ap gh_mt7688_toolchain /opt/buildroot-gcc342
$ git clone https://github.com/steward-fu/gh_mt7688_uboot.git
$ cd gh_mt7688_uboot
$ make

1. 格式化USBDisk成FAT32
2. 複製uboot.bin到USBDisk(lks7688.ldr)
3. 插入USBDisk到MT7688 USB host port
4. 按住WiFi和MPU按鈕，接著鬆開MPU按鈕
5. 20秒後鬆開WiFi按鈕(第二次WiFi LED亮時)
