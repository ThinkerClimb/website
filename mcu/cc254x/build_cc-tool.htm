TI CC245x

Build CC-Tool

司徒會想要使用TI CC254x開發板(Smart RF Board)的最主要原因，就是希望可以開發一個專屬N900的Bluetooth遊戲手把，不過，可惜N900目前沒有支援Bluetooth 4.0 BLE，因此，好像有點多買的感覺，不過即然都買了，為了避免浪費，司徒只好整理一些教學文章，分享給需要的使用者。

為了可以在N900和Pandora上開發TI CC254x相關程式，司徒上網尋找是否可以在Linux透過CC-Debugger燒錄CC254x的工具，結果發現這個CC-Tool工具，而且還是開源專案，真是相當好，CC-Tool可以透過CC-Debugger燒錄TI CC254x，因此，編譯CC-Tool就變成是首要任務，所幸沒有太複雜，步驟如下說明： 

$ sudo apt-get install libboost-all-dev
$ git clone https://github.com/steward-fu/gh_cc-tool.git
$ cd gh_cc-tool
$ ./configure
$ make clean
$ make
$ sudo make install
