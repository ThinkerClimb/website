開發環境

Ref. http://plit.de/asem-51/

雖然SDCC是一款相當不錯的編譯器，不過，當編譯純Assembly檔案時，則是相對複雜，因為很多Section以及屬性需要定義，大大增加編輯難度，因此，司徒最終使用ASEM-51編譯器，不過鑑於ASEM-51編譯器無法在Debian(ARM版本)執行，因此，司徒還使用第二款編譯器MCU8051IDE，全部程式都會在這兩個編譯器上編譯測試。

編譯器ASEM-51 

$ cd
$ wget http://plit.de/asem-51/asem51_1.3-2_i386.deb
$ sudo dpkg -i asem51_1.3-2_i386.deb
$ asem

MCS-51 Family Macro Assembler ASEM-51 V1.3

usage:     asem [options] <source> [<object> [<listing>]]

options:   -i   --includes=path1:path2:path3
           -d   --define=symbol[:value[:type]]
           -o   --omf-51
           -c   --columns
           -v   --verbose
