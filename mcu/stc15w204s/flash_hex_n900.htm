STC STC15W204S

如何透過N900使用UART燒録程式

司徒將在這篇文章中，介紹如何使用N900燒録STC15W204S單晶片，需要的東西如下：
STC15W204S GPIO開發板

N900並安裝Easy Debian環境

USB轉TTL RS-232(PL2303)

準備完成後，連接到N900並切換成USB Host Mode

由於司徒是使用stcgal程式，因此還需要安裝python3、python3-setuptools套件： 

$ sudo apt-get install python3 python3-setuptools

接著下載stcgal程式並安裝 

$ git clone https://github.com/grigorig/stcgal.git
$ cd stcgal
$ ./setup.py build
$ sudo ./setup.py install

首先査看PL2303裝置使否可正常抓到

執行燒録指令： 

$ sudo stcgal -P stc15 main.hex

執行到此命令時，重新Reset開發板

燒録完成
