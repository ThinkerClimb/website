安装環境

SDL(Simple DirectMedia Layer)是目前主流的遊戲開發Library，提供了一個便利且統一的接口讓使用者開發遊戲使用，如果不透過SDL來製作遊戲，首先遇到的問題就是複雜的溝通過程(與底層驅動程式)，並且會遇到每個平台(ARM、MIPS、x86)的溝通方式可能都不一樣，因此，當要移植遊戲或者模擬器到其它平台時，有可能會遇到需要重新改寫的問題，當然優點不只司徒說的這一丁點而已，因此，如果想要開發遊戲或者想要玩玩懷舊模擬器的使用者，司徒建議好好學習SDL的撰寫技巧，畢竟這是目前懷舊電玩的主流平台。

司徒目前撰寫測試SDL程式的平台是使用PC(Debian x64)，因此，使用者可以使用如下命令安裝SDL套件，而編輯的工具則是使用vim工具 

$ sudo apt-get install libsdl1.2-dev libsdl-image1.2-dev libsdl-mixer1.2-dev libsdl-gfx1.2-dev libsdl-sound1.2-dev
