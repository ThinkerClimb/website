Build GMenu2X 0.12

[Github] https://github.com/steward-fu/gh_n900_emulator.git

由於GMenu2X是一款相當適合小掌機使用的主操控介面，如：Wiz、Caanoo、A320、GCW-Zero上都可以看到身影，因此，司徒也想將它移植到N900身上，希望N900也擁有更佳的遊戲操控介面，編譯步驟如下說明： 

$ git clone https://github.com/steward-fu/gh_n900_emulator.git
$ cd gmenu2x-0.12
$ cp sdl_ttf-2.0.11/SDL_ttf.o .
$ cp sdl_ttf-2.0.11/SDL_ttf.h /usr/include/SDL/
$ make

完成
