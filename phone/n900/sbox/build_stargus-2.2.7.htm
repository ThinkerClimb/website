Build Stargus 2.2.7

[Github] https://github.com/steward-fu/gh_n900_emulator.git

目前Maemo系統上，最新的Stargus模擬器版本是2.2.7版，使用者可以參考如下步驟編譯： 

$ git clone https://github.com/steward-fu/gh_n900_emulator.git
$ cp stargus_include/* stargus_2.2.7/
$ cd stargus_2.2.7
$ mkdir build
$ cd build
$ cmake ../
