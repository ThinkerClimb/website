重新格式化分區

[Github] https://github.com/steward-fu/gh_n900_deb-packages.git

N900預設的分區大小是：root 256MB、MyDocs 29GB、/home 2GB以及swap 768MB，容量以MyDocs(FAT32)為主要存放空間，不過司徒存放的資料卻都是以Linux(ext3)為主，因此，司徒便重新規劃分區成：MyDocs 1GB、/home 30GB和swap 768MB，MyDocs主要存放MStarDict字典檔案和導航地圖檔案，因此，1GB的容量就足夠存放，剩餘容量則給/home分區使用，至於swap容量，司徒還是維持768MB，重新分區後的容量大小如下表： 

Disk /dev/mmcblk0: 32.0 GB, 32015122432 bytes
4 heads, 16 sectors/track, 977024 cylinders
Units = cylinders of 64 * 512 = 32768 bytes

Device Boot     Start   End      Blocks    Id   System
/dev/mmcblk0p1  2       30520    976608    c    Win95 FAT32 (LBA)
/dev/mmcblk0p2  30521   950401   29436192  83   Linux
/dev/mmcblk0p3  950402  974977   786432    82   Linux swap

Command (m for help):

操作步驟如下：
1. 安裝bootmenu套件
2. 執行bootmenu安裝程式
3. 下載司徒的Github並且把fdisk放到/sbin/資料夾底下
4. 重新開機進入bootmenu並Backup rootfs & opts到SDCard
5. 重新開機進入maemo系統並開啟Terminal 

$ sudo chmod 0777 /sbin/fdisk
$ sudo fdisk /dev/mmcblk0

分區參考如上表格，修改分區後，請重新開機進入bootmenu，接著從SDCard Restore rootfs & opts後，再度重開機進入maemo即可。
