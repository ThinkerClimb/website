Sony VGN-P25

調整CPU速度(Ubuntu)

Sony VGN-P25機器是司徒使用過的UMPC中，最適合桌上使用的尺寸(不適合掌上使用)，不過唯一的缺點就是續航力問題，未調整過的正常使用時間(Ubuntu系統)大約是1.5小時(關閉Wifi)，這個使用時間算是相當差的表現，因此，除非更換大容量電池，不然就是需要好好調整一下CPU速度以及顯示屏的亮度，這樣才有機會讓續航力增加，而司徒今天就要介紹一下如何在Ubuntu系統下調整CPU速度，藉此讓續航力更持久。

高效能(1.86GHZ) 

# echo performance > /sys/devices/system/cpu/cpu0/cpufreq/scaling_governor
# echo performance > /sys/devices/system/cpu/cpu1/cpufreq/scaling_governor

省電(800MHZ) 

# echo powersave > /sys/devices/system/cpu/cpu0/cpufreq/scaling_governor
# echo powersave > /sys/devices/system/cpu/cpu1/cpufreq/scaling_governor
