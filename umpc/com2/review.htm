Sony Mylo COM-2

介紹

Mylo COM-2機器也是司徒在大板電器街發現的機器，當初購買的動機也是因為好奇心，想說可以研究看看這一台機器究竟有什麼功能，後來司徒才發現COM-2機器的程式碼已經被Sony Release出來，因此，購買這一台機器除了好奇心以外，更多了一個開源系統的研究功能。

簡要規格
CPU i.MX31L(ARM1136F-JS) 532MHz
RAM 128MB
Flash 1GB
Screen 3.5" 800x400 電阻觸控
MS Pro Duo 支援到256GB
Keyboard 四排QWERTY
USB USB(支援Client)
WLAN Wi-Fi 802.11 b/g
Battery 3.7V 1450mA
Dimension 130mm x 64mm x 20mm
Weight 193g
