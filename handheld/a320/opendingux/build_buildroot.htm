Dingoo A320 ⇒ OpenDingux

Build Buildroot

OpenDingux系統有釋出buildroot的config檔案，方便使用者添加或者刪除相關元件，如果使用者有興趣自行編譯系統，可以參考如下方式： 

$ git clone https://github.com/steward-fu/a320_toolchain.git
$ sudo mv a320_toolchain/mipsel-linux-uclibc /opt/toolchain
$ export PATH=$PATH:/opt/toolchain/usr/bin/
$ git clone https://github.com/steward-fu/a320_opendingux.git
$ cd a320_opendingux/buildroot-201211
$ make a320_defconfig
$ make

編譯後的RootFS檔案位於：output/images/rootfs.squashfs
