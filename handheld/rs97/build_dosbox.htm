Build DOSBox

編譯步驟如下： 

$ cd
$ git https://github.com/dmitrysmagin/dosbox-gcw0
$ cd dosbox-gcw0
$ ./autogen.sh
$ export TARGETMACH=mipsel-linux
$ export BUILDMACH=i686-pc-linux-gnu
$ export CROSS=mipsel-linux
$ export CC=${CROSS}-gcc
$ export LD=${CROSS}-ld
$ export AS=${CROSS}-as
$ export CXX=${CROSS}-g++
$ SDL_CONFIG=/home/steward/Downloads/buildroot-2017.02.9/output/host/usr/mipsel-buildroot-linux-uclibc/sysroot/usr/bin/sdl-config ./configure --host=$TARGETMACH
$ vim src/Makefile +249
  LIBS = -lSDL_sound -lasound -lm -ldl -lpthread -L/opt/retrogame/usr/mipsel-buildroot-linux-uclibc/sysroot/usr/lib -lSDL -lpng -lz -lSDL_net -lSDL_gfx -lSDL_image

$ make

P.S. SDL_CONFIG請依照你的toolchain位置指定。
